## Scoping Lecture

- Statement of Work (SOW) (living document of services):
  - Scope of work
  - Description
  - Cost
  - How to build
  - To update the document:
    - Change order
    - Extension
- Master Service Agreement (MSA)
  - Relationship with the client
- Scope
  - This is what we want
  - This is how long
  - This is what we will get
